Japan Pictograms on your Website

The pictograms of Experience Japan Pictograms, made easy to use for your website.
Provided by Yann Klein.

Summary

1. Setup 2. Usage 2. a. Sizes 2. b. Pictograms

Setup

Add this link to your HTML file <head>:

<link rel="stylesheet" href="https://yannklein.github.io/japan-pictograms/japan-pictograms.css">

Usage

Add icons following this pattern, refer to the below table for the NAME:

<span class="jpic jpic-NAME"></span>

Size and color

Japan Pictograms is a font, use normal CSS font properties to change its size, color and other font characteristics.

Sample code Result
<span class="jpic jpic-ise-jingu" style="font-size: 64px; color: red"></span>
